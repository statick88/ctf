# InvestigaOSINT Platform Documentation

## 1. Platform Overview

InvestigaOSINT is the **first Brazilian platform focused on OSINT (Open Source Intelligence) CTFs (Capture The Flag)** challenges. It aims to revolutionize OSINT learning in Brazil by providing practical, scenario-based challenges that simulate real-world digital investigations.

Key features:
- Dedicated platform for OSINT training
- Practical challenges based on real-world Brazilian cases
- Updated articles on OSINT, SOCMINT, GEOINT, and other techniques
- Essential investigation tools directory
- Video tutorials for beginners and advanced users
- Community-focused learning environment

## 2. Registration Process

### How to Register:
1. Visit the InvestigaOSINT website at https://investigaosint.io/
2. Click on "CTFs" in the navigation menu (or go directly to https://investigaosint.io/ctfofc/)
3. On the CTF page, click the "Cadastre-se" (Register) button
4. This will redirect you to the user registration page (https://investigaosint.io/registro-de-usuario/)
5. Complete the registration form with your details
6. After registration, you'll be able to access the CTF challenges

**Note**: The platform uses WordPress with Ultimate Member plugin for user management.

## 3. Login Process

### How to Login:
1. Visit the login page directly at https://investigaosint.io/login/
2. The login form is powered by Forminator plugin
3. Enter your username or email address
4. Enter your password
5. Click the login button to access your account
6. If you forgot your password, use the "Forgot your password?" link to reset it

**Features**:
- Remember me functionality
- Password reset option
- Secure login with reCAPTCHA protection
- Integration with Ultimate Member plugin

## 4. CTF Challenge Interface

### Accessing Challenges:
1. Login to your InvestigaOSINT account
2. Navigate to the CTF section (https://investigaosint.io/ctfofc/)
3. Browse available challenges

### Challenge Display:
- Challenges are presented in a carousel format on the CTF page
- Each challenge has a featured image and title
- Categories visible in the carousel include:
  - Desafio 3 (Challenge 3)
  - Whois
  - Italy Mountains (Geolocation challenge)
  - Email OSINT
  - Pessoa desaparecida (Missing person)
  - Traição (Betrayal)

### Challenge Format:
- Scenario-based investigations
- Focus on real-world Brazilian cases
- Progressive difficulty levels
- Various OSINT techniques:
  - Geolocation (GEOINT)
  - Social media intelligence (SOCMINT)
  - Email investigation
  - Domain/Whois research
  - Image analysis
  - Missing person investigations

## 5. Flag Submission Process

While the exact submission process isn't explicitly detailed in the available HTML, based on standard CTF platforms:

1. Solve the challenge by finding the flag (usually a string format like `FLAG{...}`)
2. Navigate to the challenge page
3. Locate the flag submission form
4. Enter the flag
5. Submit to receive points

**Expected Features**:
- Points awarded for correct submissions
- Leaderboard tracking
- Time-based scoring
- First blood bonuses
- Challenge completion badges

## 7. Known Features and Limitations

### Features:
- **Comprehensive OSINT Tools Directory**: Curated list of essential tools including Maltego, Spiderfoot, Shodan, Metagoofil, Wayback Machine, Epieos, WhatsMyName, and Whois Lookup
- **Educational Content**: Articles, tutorials, and video lessons
- **Community**: Social media presence on Instagram, LinkedIn, Facebook, and YouTube
- **Brazilian Focus**: Challenges and content tailored to Brazilian context
- **Multimedia Challenges**: Image, video, and scenario-based investigations
- **Regular Updates**: New challenges and content added periodically

### Limitations:
- **Language Barrier**: Platform is entirely in Portuguese (may be challenging for non-Portuguese speakers)
- **Limited Challenge Information**: Detailed challenge descriptions not publicly available without registration
- **No Public Leaderboard**: Leaderboard access likely requires login
- **WordPress Limitations**: As a WordPress-based platform, it may have some performance and scalability constraints
- **Content Availability**: Some features may be restricted to registered users only

### Technical Details:
- **Platform**: WordPress 6.9
- **Theme**: Hello Elementor
- **Page Builder**: Elementor 3.35.6
- **User Management**: Ultimate Member 2.10.5
- **Form Builder**: Forminator
- **Analytics**: Google Analytics (MonsterInsights)
- **Cookie Consent**: Cookie Law Info
- **Security**: reCAPTCHA protection
